# tmllm
Solutions for Test & Measurement controlling by LLM

## WARNING: LLMs is lack awareness of the real world, that could potentially KILL YOU.

During the test, LLMs exhibited a lack of intuition. For example, in commands measuring fall time, the LLMs would overscale the oscilloscope to the point that the entire capture window could not show the falling edge, causing the LLM to enter an infinite loop. Therefore, it is recommended that users always maintain close monitoring and vigilance of the instrument and provide sufficiently human support in the loop.

**LLM Usage Disclosure**

The source code in this repo is partially generated by `DeepSeek V4 Pro`, then reviewed and approved by me. The descriptive document is fully written by hand, and the document of parameters and functions is generated from source code.

For drivers.py, it's usually generated from manual by LLM and fully reviewed by the maintainer. However, those function is completely tested by hand and approved.
